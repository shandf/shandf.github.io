How to Change the Web Port of GitLab

Step 1: GitLab Shell

Change the GitLab Shell config file: /home/git/gitlab-shell/config.yml:

gitlab_url: "http://***.***.***.***:83/"

Step 2: GitLab

Change the GitLab config file: /home/git/gitlab/config/gitlab.yml:

host: ***.***.***.***
port: 83

Step 3: ngnix

Change the nginx config file: ` /etc/nginx/sites-available/gitlab`:

listen *:83 default_server;
