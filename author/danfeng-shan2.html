openSUSE 安装 VirtualBox 虚拟机

使用 sudo zypper in virtualbox 之后出现问题，提示运行命令 /etc/init.d/vboxdrv setup 安装内核模块，可是输入命令之后产生如下输出:

Recompiling VirtualBox kernel module, NOT. It has been packaged.

阅读 /etc/ini.d/vboxdrv 内容之后，发现 setup 函数只做一件事：输出上面的内容。所以，可以肯定，应该是安装时出现的问题。 卸载所有与 virtualbox 相关的包，并安装内核源码 "kernel-desktop-devel"，下载 virtualbox 官网上的 rpm 包，安装，成功解决问题。

如何安装 fcitx

以ubuntu为例，说明安装企鹅输入法的步骤 首先，要先卸载 ibus:

sudo apt-get remove ibus

然后，通过 "apt-get" 命令，安装企鹅输入法以及谷歌拼音:

sudo apt-get install fcitx fcitx-googlepinyin

安装完之后，在终端下可能还不能切换，因此，要修改终端的 .bashrc 文件，向其添加如下内容:

export GTK_IM_MODULE=xim
export QT_IM_MODULE=xim
export XMODIFIERS="@im=fcitx"

How to Change the Web Port of GitLab

Step 1: GitLab Shell

Change the GitLab Shell config file: /home/git/gitlab-shell/config.yml:

gitlab_url: "http://***.***.***.***:83/"

Step 2: GitLab

Change the GitLab config file: /home/git/gitlab/config/gitlab.yml:

host: ***.***.***.***
port: 83

Step 3: ngnix

Change the nginx config file: ` /etc/nginx/sites-available/gitlab`:

listen *:83 default_server;
