Windows 下交换 Caps 键与 Ctrl 键

打开注册表编辑器

win + R 打开运行，输入 regedit

找到目标文件夹

HKEY_LOCAL_MACHINE -> System -> CurrentControlSet -> Control -> KeyBoard Layout

新建注册表项

右击该文件夹，点击 新建 -> 二进制值, 修改注册表项的名字为 Scancode Map, 然后右击该注册表项，点击 修改二进制数据, 手动输入如下数据:

0000 00 00 00 00 00 00 00 00
0008 03 00 00 00 3A 00 1D 00
0010 1D 00 3A 00 00 00 ...

How to Change the Web Port of GitLab

Step 1: GitLab Shell

Change the GitLab Shell config file: /home/git/gitlab-shell/config.yml:

gitlab_url: "http://***.***.***.***:83/"

Step 2: GitLab

Change the GitLab config file: /home/git/gitlab/config/gitlab.yml:

host: ***.***.***.***
port: 83

Step 3: ngnix

Change the nginx config file: ` /etc/nginx/sites-available/gitlab`:

listen *:83 default_server;
