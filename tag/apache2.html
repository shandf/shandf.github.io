Enable Userdir (public_html) in Ubuntu 12.04

Firstly, enable module userdir:

sudo a2enmod userdir

Then, change apache config file /etc/apache2/mods-enabled/userdir.conf:

# line 6-7
AllowOverride All
Options ExecCGI

Finally, change the apache config file /etc/apache2/mods-enabled/php5.conf:

# line 23: comment
# /etc/apache2/mods-enabled/php5.conf

Restart apache server:

sudo service apache2 restart
